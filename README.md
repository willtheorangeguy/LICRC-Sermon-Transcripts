## Key Features

- Ability to download all pre-created transcripts.
- Pull audio of all LICRC sermons.
- Create transcripts for every episode using OpenAI's Whisper.
- Generate a summary from the transcript using Ollama and Llama.
- Correct spelling and grammatical errors using LanguageTool.

## How To Use

**To clone and run your own copy of this website**, you'll need [Git](https://git-scm.com/downloads), [Ollama](https://ollama.com/) and a bunch of Python libraries installed on your computer. If you would rather not use Git, you can just download the code from GitHub [above](https://github.com/willtheorangeguy/LICRC-Sermon-Transcripts/archive/refs/heads/main.zip). From your command line:

```bash
# Clone this repository
$ git clone https://github.com/willtheorangeguy/LICRC-Sermon-Transcripts.git

# Go into the repository
$ cd WAN-Transcripts

# Install Dependencies + Run
$ pip install -r requirements.txt
$ pip install git+https://github.com/openai/whisper.git
$ pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
$ python main.py <year>
```

If support is required, please open a **[GitHub Discussion](https://github.com/willtheorangeguy/LICRC-Sermon-Transcripts/discussions/new)** or join our **[Discord](https://discord.gg/mgbda4fesN)**.
